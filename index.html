В диалоговом окне Параметры страницы выполните любое из следующих действий.



Чтобы изменить размер бумаги, выберите его значение из раскрывающегося списка Размер.

Чтобы изменить подачу бумаги, выберите имя лотка или податчика из раскрывающегося списка Источник бумаги.


Чтобы напечатать документ вертикально, выберите параметр книжная; чтобы напечатать документ горизонтально, выберите параметр альбомная.


Чтобы изменить поля, введите ширину в любом из полей Поля.

После ввода изменений нажмите кнопку ОК.
